Por que usar QR Pro?

📊

Analytics em Tempo Real

Acompanhe scans por periodo, dispositivo, localizacao e mais. Tudo via API ou dashboard.

🔄

QR Dinamico

Mude o destino do QR sem reimprimir. Ideal para campanhas, menus e links que evoluem.

⚡

API RESTful

Integracao simples com JWT ou API Key. Documentacao OpenAPI/Swagger completa.

🎨

Personalizacao

Cores, tamanho, correcao de erro. Em breve: logo central e estilos avancados.

🔒

Seguranca

JWT HS512, API Keys com hash SHA-256, rate limiting distribuido com Redis.

🏗

Arquitetura Hexagonal

Codigo limpo, testavel e desacoplado. Facil de manter e escalar.

Pronto para integrar?

Comece gratis. Escale conforme seu crescimento. Sem cartao de credito.
