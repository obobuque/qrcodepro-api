API

Integração em minutos

RESTful, documentada e pronta para produção. Teste no Swagger UI.

cURL
JavaScript
Python
# Gerar QR estático
curl -X POST "https://qrcodepro-api.onrender.com/api/v1/public/qr/static" \
  -H "Content-Type: application/json" \
  -d '{"content":"https://seusite.com","size":300}'

# Criar QR dinâmico (requer auth)
curl -X POST "https://qrcodepro-api.onrender.com/api/v1/qr/dynamic" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"initialUrl":"https://promo.com","size":300}'

# Listar seus QR codes
curl -X GET "https://qrcodepro-api.onrender.com/api/v1/qr" \
  -H "Authorization: Bearer <token>"
